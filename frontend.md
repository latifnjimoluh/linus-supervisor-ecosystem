The Next.js application contains skeleton routes that still require implementation of UI and business logic. Key interfaces to build include:

- User management (list, add, edit, detail)
- Role and permission management
- Password reset history
- System logs
- General settings and notifications
- Script templates and testing
- Monitoring pages and alert views
- User profile
- Server management (list, add, edit, detail)
- Script preview and deployment
- AI assistant tools
- Virtual machines and conversion workflows
- Terraform operations
- AI tools and cache management

These routes are scaffolded under `frontend/app/(app)` and can be completed incrementally.
